## Project Description

**Problem**: Finding reliable local service providers (plumbing, cleaning, tutoring) is fragmented, time-consuming, and often involves distant providers increasing carbon footprint.

**Solution**: ServiceSphere is a full-stack marketplace connecting neighborhood customers with verified local providers. Browse, book, manage services seamlessly.

**Key Features**:
- User/Provider registration & authentication (JWT)
- Search/filter services by category, location, price
- Real-time booking with date/time picker
- Dashboard for bookings/reviews
- Responsive design (mobile-first)

**Tech Stack**:
- **Frontend**: React + Vite + Tailwind CSS + React Router
- **Backend**: Node.js + Express + Mongoose
- **Database**: MongoDB
- **Deployment**: Vercel (FE) + Render (BE)

**Impact**: Reduces travel (local focus → lower emissions), supports small businesses, builds community trust via reviews/verification.
