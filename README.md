## Project Description

### Problem
Finding reliable local service providers (plumbing, cleaning, tutoring) is fragmented, time-consuming, and often involves distant providers, increasing cost and carbon footprint.

### Solution
ServiceSphere is a full-stack marketplace that connects customers with verified nearby service providers. Users can search, compare, and book services instantly.

### Key Features
- 🔍 Search services by **category & location**
- 🧑‍🔧 Verified & top-rated providers
- 📅 Easy **booking system**
- 👤 User & provider dashboards
- ⭐ Ratings and reviews
- 📱 Fully responsive UI

### Tech Stack
- **Frontend:** React + Vite + Tailwind CSS + React Router  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Deployment:** Vercel (Frontend) + Render (Backend)  

### Impact
- 🌱 Promotes **local services → reduces travel emissions**
- 💼 Supports **small businesses & freelancers**
- 🤝 Builds **trust via reviews & verification**
